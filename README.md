This repository is for practice p5.js.

Spec
====

* Node (tested w/ v7.10.0)

Preparation
-----------

`npm install`

Build
=====

Global mode (gulp)
------------------

* Development: Edit `src/sketch.js` directly
* Build: Nothing
* Watch: `gulp watch`

Instance mode (webpack)
-----------------------

* Development: Edit `es6_src/main.js` w/ webpack build
* Build: `npm run build`
* Watch: `npm run start`
